If you want to commit your code into the GitHub, please follow below steps:

1) Goto that project location (Like C:/, D:/, E:/)
2) In Git bash, Enter the command: git init [It is Initialize Git - Only first time)
3) git add .
4) git commit -m "initial commit"
5) git remote add origin https://github.com/KunalS007/cypress-automation.git
6) git remote -v
7) git branch -M main
8) git push -u origin main
